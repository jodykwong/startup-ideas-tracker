🎯 现在只需 3 步完成部署

  1. 创建 GitHub 仓库
    访问 https://github.com/new
    仓库名: startup-ideas-tracker
    设置为 Public，不要添加任何文件
  2. 运行自动化脚本
    在项目目录执行: ./auto-deploy-setup.sh
    脚本会自动完成所有 Git 和依赖配置
  3. 连接 Vercel
    访问 Vercel Dashboard 导入仓库
    配置环境变量，一键部署完成

⚙️环境变量配置

在 Vercel 项目设置中添加以下环境变量：

# Supabase 配置 PUBLIC_SUPABASE_URL=https://your-project.supabase.co PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key # Gemini AI 配置 GEMINI_API_KEY=your_gemini_api_key # 环境配置 NODE_ENV=production
⚠️ 重要提示: 确保在 GitHub Secrets 中也配置相同的环境变量，以支持自动部署功能。
